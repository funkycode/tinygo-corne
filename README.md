# tinygo-corne

## to build:
```
# build for right half
tinygo build -target nicenano -size short -o corne_right_nano.uf2 github.com/funkycode/tinygo-corne/nicenano/right

# build for left half
tinygo build -target nicenano -size short -o corne_left_nano.uf2 github.com/funkycode/tinygo-corne/nicenano/left
```
